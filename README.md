# 💎 EOS — Premium Web3 Landing Page

[![AIOX Framework](https://img.shields.io/badge/Framework-AIOX-blueviolet?style=flat-square)](https://github.com/ssnowzx/aiox)
[![React](https://img.shields.io/badge/React-18+-61DAFB?style=flat-square&logo=react)](https://reactjs.org/)
[![Tailwind CSS v4](https://img.shields.io/badge/Tailwind_CSS-v4.0-38B2AC?style=flat-square&logo=tailwind-css)](https://tailwindcss.com/)
[![TypeScript](https://img.shields.io/badge/TypeScript-5+-3178C6?style=flat-square&logo=typescript)](https://www.typescriptlang.org/)

Uma Landing Page de ultra-fidelidade para o ecossistema Web3, focada em conversão e micro-detalhes visuais. O projeto utiliza a arquitetura **Tailwind CSS v4** nativa para processamento de estilos de alto desempenho.

---

## ✨ Key Features

- **🚀 0.6px Precision:** Bordas sub-pixel e divisores ultra-finos para um visual "luxury tech".
- **🌫️ Glassmorphism 2.0:** Uso extensivo de `backdrop-blur-xl` e camadas de opacidade gradual.
- **🎬 Immersive Background:** Vídeo de fundo em loop com overlays cinematográficos e transições atmosféricas.
- **⚡ Technical Grid (Xadrez):** Textura de fundo técnica para o rodapé, conferindo um aspecto de "blueprint".
- **📝 Genesis Waitlist:** Modal de conversão animado com Framer Motion para captura de leads.
- **📱 Single Page Experience:** Jornada fluida de rolagem única com navegação por âncoras inteligentes.

---

## 🛠️ Tech Stack

- **Core:** [React 18](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/) (Next-gen architecture)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Typography:** General Sans (Fontshare)

---

## 🚀 Quick Start

1. **Instale as dependências:**
   ```bash
   npm install
   ```
2. **Inicie o servidor:**
   ```bash
   npm run dev
   ```
3. **Build para Produção:**
   ```bash
   npm run build
   ```
