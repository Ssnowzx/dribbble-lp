# Story 3.1: Bento Grid Layout & Glow Effect

## 1. Context & Goals
**Epic:** 3 - Feature Showcasing
**Objective:** Criar uma grade Bento responsiva com micro-interações de luz (Glow Radial) que seguem o cursor do mouse, proporcionando uma experiência imersiva e luxuosa.

---

## 2. Requirements
- **Visual:** Layout assimétrico, bordas de 0.6px, cantos de 24px.
- **Interação:** Gradiente radial (glow) que segue o mouse em cada card.
- **Performance:** Uso de `useMotionValue` para evitar re-renders.

---

## 3. Acceptance Criteria (AC)
- [ ] **AC 3.1.1:** Criar componente `<BentoGrid />` e subcomponente `<BentoCard />`.
- [ ] **AC 3.1.2:** Implementar o efeito de luz que segue o cursor (Framer Motion).
- [ ] **AC 3.1.3:** Garantir responsividade (grid de 3 colunas no desktop, 1 no mobile).
- [ ] **AC 3.1.4:** Adicionar ícones Lucide e textos refinados.
