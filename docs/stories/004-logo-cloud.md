# Story 2.3: Logo Cloud Infinite Marquee

## 1. Context & Goals
**Epic:** 2 - Layout & Hero Experience
**Objective:** Implementar uma faixa de logos (Logo Cloud) com animação de rolagem infinita e suave, reforçando a autoridade da marca com parceiros Web3/SaaS.

---

## 2. Requirements
- **Visual:** Logos em escala de cinza/branco com 60% de opacidade.
- **Animation:** Marquee infinito (CSS Keyframes). Pausa suave no hover.
- **Reference:** `docs/architecture/technical-design.md`

---

## 3. Acceptance Criteria (AC)
- [ ] **AC 2.3.1:** Criar componente `<LogoCloud />` em `src/components/blocks/LogoCloud/`.
- [ ] **AC 2.3.2:** Implementar animação de marquee infinita via CSS.
- [ ] **AC 2.3.3:** Adicionar suporte a hover para pausar a animação.
- [ ] **AC 2.3.4:** Logos devem ser responsivos e bem distribuídos.
